# Installation

## OS X & Linux:

### Direct
```sh
# Install zipped project
wget -O wp_recovery.zip https://codeload.github.com/Noam-Alum/make_backup/zip/refs/heads/main
```

### GitClone
```sh
# Clone
git clone https://github.com/Noam-Alum/wp_recovery.git
```
